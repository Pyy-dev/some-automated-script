### Seeyon_OA

```
用法
optional arguments:
  -h, --help          show this help message and exit
  -u URL, --url URL     please input URL 
  -f FILE, --file FILE  please input URL file 
  -n NAME, --name NAME  please input shell name
```

可以单个也可以批量，shell的名字可任意修改，默认采用冰蝎的马，可以自行修改
